# How to Create a Shared School Calendar

Getting into the classroom consistently is key to building strong teacher-admin relationships, providing high-quality feedback and coaching, and ultimately growing instruction and learning.



**HELPFUL HINTS FROM BULLSEYE USERS:**



[https://player.vimeo.com/video/504936103](https://player.vimeo.com/video/504936103)



Our partners at Etoile Academy have built systems optimized to get administrators into the classroom consistently and seamlessly. One of the keys to success is having an easily accessible digital school-wide schedule. This allows Etoile's admin team to easily coordinate who they should be observing, and when and where to visit. Check out how **Etoile Academy Founder and Superintendent Kayleigh Colombero** approaches school-wide scheduling in the video above!



Create a school-wide calendar by following the steps below:

1. Use Google Calendar or Microsoft Outlook to create a school-wide shared calendar

2. Ensure it is easily accessible to every leader (Helpful Links: [Google Calendar](https://support.google.com/calendar/answer/37083?hl=en#zippy=%2Cmark-your-calendar-as-public) // [Microsoft Outlook](https://support.microsoft.com/en-us/office/share-an-outlook-calendar-with-other-people-353ed2c1-3ec5-449d-8c73-6931a0adab88))

3. Create a calendar event for each lesson which should include subject, the teacher who will be leading instruction, and the location of the classroom or link to that virtual lesson
