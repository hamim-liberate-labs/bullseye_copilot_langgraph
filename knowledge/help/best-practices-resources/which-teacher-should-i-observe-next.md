# Which Teacher Should I Observe Next?

You've blocked off time for observation but which classroom should you observe? For the sake of time, it's important to have systems in place that can help make that decision quick, easy, and intentional. Below are two possible strategies:



1. Use the Bullseye Frequency Report. This report shows the number of times a teacher has been observed in the Last Week, Month, and All Time so you can ensure each teacher is accounted for and receiving the support they need.



[https://player.vimeo.com/video/668241894](https://player.vimeo.com/video/668241894)





2. Use the Notecard Technique:



1. Ask each teacher to create a notecard, writing their name on one side and their schedule on the other side.

2. Collect all notecards and create a stack.

3. When it's time for another classroom, pick a notecard off the top, check the schedule to confirm that time works for the selected teacher.

4. If the time works, move that notecard to the bottom of the stack, and complete a walkthrough. If the time doesn’t work, continue selecting off the top until a teacher & time works -- but make sure to place the notecards that don't work back on top of the stack for next time.
