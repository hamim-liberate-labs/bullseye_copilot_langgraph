# Issue with Sessions

**Issues within Sessions**

**Deleting a Published Session**
-To delete a session, you must be the observer who created it.

-Follow these simple steps to remove a session:

1. Access Your Sessions - click Sessions in the sidebar menu.
2. Locate the Session - browse the list or use the search bar to find the session you want to delete.
3. Open the Session - click the session name to open it in a new page.
4. Revise/Continue the Session
-**If the session is published,** click the Revise Session button.
-**If the session is a draft**, click the Continue Session button
4. Edit and Delete - click the pencil icon at the top left beside the session name, scroll to the very bottom of the page and click Delete.
5. Confirm Deletion - when prompted, click OK to confirm you want to delete the session.

**How to Publish or Delete a Draft Session**
-Every session you create is automatically saved as a draft.

-If you need to pause and return later, your progress will remain intact.

-When you're ready, you can either publish the session or delete it. Here's how:


To Publish a Draft Session
1. Navigate to your Dashboard or the Sessions Page.
2. Click on the draft session you’d like to publish.
3. Select Continue Session.
4. Add any additional scores or notes.
5. Click Publish to finalize the session.

To Delete a Draft Session
1. Navigate to your Dashboard or the Sessions Page.
2. Click on the draft session you’d like to delete.
3. Select Continue Session.
4. Click the blue pencil icon in the upper left corner.
5. Select Delete Session to remove it permanently.
