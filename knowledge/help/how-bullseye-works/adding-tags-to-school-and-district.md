# Adding Tags to School and District

## School Tags

School administrators can manage tags by navigating to **School Admin > School Tags** from the menu in the upper-left corner of the dashboard. On the School Tags page, you can view, edit, or delete existing tags.

To add new tags:

1. Click **Add Tag**.

2. Enter the tag name and press **Enter/Return**.

3. Repeat to add multiple tags at once.

4. Click **Add Tags** to save.

Once added, these tags will be available to use when conducting a session.

## District Tags

District administrators can manage district-wide tags by navigating to **District Admin > District Tags** from the menu in the upper-left corner of the dashboard. Here, you can view, edit, or delete existing district tags.

To add new district tags:

1. Click **Add Tag**.

2. Enter the tag name and press **Enter/Return**.

3. Repeat to add multiple tags at once.

4. Select which schools should receive the new tags.

5. Click **Add Tags** to save.

District tags can be assigned to one, several, or all schools in your district, helping provide context for observation sessions and enabling more detailed data analysis across schools.
