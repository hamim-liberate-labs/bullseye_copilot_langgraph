**Where do coaching plans live in Bullseye?**

• Coaching plans live in the Active Goal section of a teacher’s profile.

**How does an admin access a teacher’s coaching plan?**

1. Log into Bullseye

2. Go to the left menu

3. Click People

4. Select the teacher

5. Scroll to the Active Goal section

**How do you create a new coaching plan for a teacher?**

1. Go to People → [Teacher Name]

2. In the Active Goal section, click Start New Goal

3. Expand the text box (optional)

4. Enter the coaching plan information

5. Use template if available

6. Click minimize to save

7. Click Publish

**What happens when you click “Start New Goal”?**

• A new editable goal field opens

• If turned on, you can use a plan template

• When publishing a new goal, the previous goal will be replaced but saved in the history

**Can coaching plans be aligned to the instructional framework?**

Yes, they can be!

• Admin can associate an objective with the coaching plan

• Ensures alignment to rubric/framework

**Where will an active coaching plan show up?**

In two places!

• At the top of every session for that teacher

• In the teacher’s profile under Active Goal

**How do admin edit an existing coaching plan?**

1. Go to People → [Teacher Name]

2. Click Edit Current Goal

3. Make changes

4. Minimize to save

5. Click Publish

**What happens to previous coaching plans or goals when a new one is created?**

- The new goal replaces the previous goal

- The View History button shows goal history

**How can admin view previous coaching plans or goal changes?**

• Go to teacher’s profile

• Click View History

**How are Next Steps connected to coaching plans?**

Admin can add Next Steps. Next Steps include:

• Title

• Description

• Linked objective

• Due date

• Next Steps support coaching plan implementation
