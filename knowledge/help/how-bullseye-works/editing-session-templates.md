# Editing Session Templates

This video walks you through how to edit a session template in Bullseye. You’ll learn how to update the session title, adjust visibility and access settings, customize objectives, and modify session notes to better align with your school’s goals. Keep in mind that changes to shared templates will apply to all listed shared staff, so be sure everyone is aligned before making updates.
