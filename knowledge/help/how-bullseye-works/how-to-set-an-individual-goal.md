# How to Set An Individual Goal

Below are steps you can take or that you can share with your staff to set these up. We recommend setting aside 15 minutes of professional development for teachers to set their goals.

1. Go to the user’s profile

  1. If you are setting a goal for yourself simply click on your name and click profile

  2. If you are creating a goal for another person, go to the people page, find the user, and click on their name

2. Click Add Goal

3. Type the goal in the text box

4. Publish
