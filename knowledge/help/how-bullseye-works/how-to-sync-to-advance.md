# How to Sync to Advance

# Syncing Bullseye with Advance

Bullseye now integrates directly with the **Advance Evaluation System**, allowing you to import formal and informal evaluations with just a few clicks. This integration helps streamline compliance with district and state requirements while keeping teachers engaged and feedback consistent.

## Syncing Bullseye to Advance Directions:

[https://player.vimeo.com/video/1172672162](https://player.vimeo.com/video/1172672162)

**Steps to Follow:**

1. Go to your Bullseye Sessions page and click Sync to Advance.

2. Drag the link that appears to your browser’s bookmark bar.

3. Open Advance in your browser.

4. Begin a new MOTP observation on a teacher and enter the session type, observer, and period information.

5. When ready, click your new Advance bookmark from the bar.

6. Enter your password: 123

7. You’ll now see a list of your Bullseye sessions. You can search by teacher name if needed.

8. Select the session you’d like to sync.

9. Once synced, your objective notes, scores, media, next steps, and session notes will automatically import into Advance.

(*Please note when syncing media,* file sizes must meet the *Advanced requirements.)*

**Tips:**

- You only need to set up the bookmark once. The integration updates automatically, no need to recreate the bookmark.

- If Next Steps or session notes are created in Bullseye on the session you’re syncing, they will automatically appear in the “Additional Notes” section in Advance.

- If an objective has a score in Bullseye, it must also include a note in order for the session to be saved or published in Advance.

## Troubleshooting

| **Issue**                         | **Possible Cause**                             | **Solution**                                                                            |
| --------------------------------- | ---------------------------------------------- | --------------------------------------------------------------------------------------- |
| **Advance session will not save** | Objective has a score but no accompanying note | Ensure all scored objectives also include a narrative note before saving or publishing. |
| **Sync not working**              | Old or outdated bookmark                       | Delete and re-add the Advance bookmark using the updated setup steps.                   |
| **Password error**                | Mistyped password                              | Double-check and re-enter the password: `123`.                                          |
